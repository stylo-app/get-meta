# Rollup Ts React Eslint boilerplate

### Development
First install dependencies:

```sh
yarn
```
or using npm
```sh
npm install
```

To run in hot module reloading mode:

```sh
yarn start
```
or using npm
```sh
npm start
```

To check for linting errors/warnings:

```sh
yarn lint
```
or using npm
```sh
npm run lint
```

### To create a production build:

```sh
yarn build
```
or using npm
```sh
npm run build
```

### Running locally

Open the file `dist/index.html` in your browser
